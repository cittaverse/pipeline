## 二、首次联系话术 (电话/微信)

### 2.1 开场白 (30 秒)

```
您好，请问是 [机构名称] 的 [联系人姓名] 吗？

我是一念万相科技的创始人 V。我们正在开展一项"AI 辅助回忆工具"
的用户体验研究，想邀请贵机构的老年朋友参与 2 周的免费试用。

请问您现在方便听我简单介绍一下吗？
```

### 2.2 核心价值 (1 分钟)

```
【研究内容】
- 2 周时间，每周 2 次、每次 30 分钟的 AI 引导回忆会话
- 在家或在贵机构场地都可以进行
- 不做医疗诊断，不做临床声称，纯用户体验研究

【参与者获益】
- 免费使用 AI 回忆工具 (市场价¥299/月)
- 获得个性化回忆录一份 (含文字稿 + 音频)
- 参与科研，帮助改进产品

【机构获益】
- 为机构老人提供创新服务
- 免费获得研究分享报告
- 社会责任形象提升
```

### 2.3 合作请求 (30 秒)

```
我们希望能与贵机构合作招募 10-15 位参与者。

合作方式很灵活：
- 方式 1: 贵机构协助转发招募信息，我们直接联系参与者
- 方式 2: 贵机构提供场地，我们上门组织会话
- 方式 3: 两者结合

整个过程中，贵机构不需要承担任何费用或风险。

请问您有兴趣进一步了解吗？
```

### 2.4 常见回应处理

**有兴趣**:
```
太好了！我稍后微信/邮件发您详细方案，您审阅后我们再约时间
细聊。请问您的微信/邮箱是？
```

**需要考虑**:
```
完全理解！这是详细方案，您先看看。我后天 (周 X) 再跟您确认
一下，您看方便吗？
```

**暂时不需要**:
```
没问题，感谢您的时间！如果以后有需要，随时联系我。
祝您工作顺利！
```

## 五、常见问题解答 (FAQ)

### Q1: 这是医疗研究吗？需要伦理审批吗？

**A**: 这是用户体验研究，不是医疗研究。
- 不做医疗诊断
- 不做临床声称
- 不涉及药物或医疗器械
- 建议做简易伦理审查，但非强制

### Q2: 参与者需要付费吗？

**A**: 完全免费。
- 工具使用免费 (市场价¥299/月)
- 个性化回忆录免费
- 无任何隐藏费用
- 参与者还获得小礼品

### Q3: 数据安全如何保障？

**A**: 我们采取多重保护措施：
- 数据加密存储 (AES-256)
- 去标识化处理 (不使用真实姓名)
- 仅研究团队可访问
- 6 个月后销毁原始音频
- 符合《个人信息保护法》要求

### Q4: 参与者可以随时退出吗？

**A**: 可以，随时退出。
- 无需理由
- 无任何惩罚
- 已收集数据可选择删除
- 不影响在机构的其他服务

### Q5: 机构能获得什么？

**A**: 机构获得：
- 免费研究分享报告
- 社会责任形象提升
- 创新服务体验
- 未来优先合作权

### Q6: 如果参与者听不懂 AI 说话怎么办？

**A**: 我们有多种支持：
- 语速可调慢
- 问题可重复
- 支持方言识别 (部分)
- 家属/工作人员可协助
- 技术支持热线随时待命

### Q7: 研究结束后，参与者还能继续使用吗？

**A**: 可以协商。
- Pilot 研究期间免费
- 研究结束后，可优惠续费
- 具体政策后续通知

## 六、合作协议模板 (简版)

```
合作备忘录

甲方：一念万相科技
乙方：[机构名称]

一、合作内容
乙方协助甲方招募 10-15 位老年参与者，参与"AI 辅助回忆工具"
用户体验研究。

二、甲方义务
1. 提供免费工具使用 (价值¥299/月)
2. 为每位参与者提供个性化回忆录一份
3. 研究结束后向乙方分享研究结果
4. 承担研究过程中的全部技术和安全责任

三、乙方义务
1. 协助招募 10-15 位参与者
2. 提供场地支持 (如需要)
3. 协助协调参与者和时间
4. 不向参与者收取任何费用

四、保密条款
1. 双方对参与者信息保密，不对外泄露
2. 研究数据仅用于学术和产品改进
3. 未经对方同意，不向第三方披露合作细节

五、知识产权
1. 研究产生的知识产权归甲方所有
2. 乙方可在宣传中使用"合作机构"名义
3. 学术论文中可致谢乙方支持

六、合作期限
2026-03-20 至 2026-04-11

七、其他
1. 本协议一式两份，双方各执一份
2. 未尽事宜，双方协商解决

甲方 (签字)：________  日期：________
乙方 (签字)：________  日期：________
```
