## 筛选逻辑 (v1.1 完整版)

### 自动筛选规则

```
Q3:  60-80 岁 → 继续;  其他 → 终止
Q6:  "完全独立" 或 "偶尔帮助" → 继续;  "全程帮助" 或 "没有手机" → 终止
Q7:  "正常" 或 "轻度" → 继续;  "中度" → 继续 + 标记人工确认;  "重度" → 终止
Q8:  "正常" 或 "轻度" → 继续;  "中度" → 继续 + 标记人工确认;  "重度" → 终止
Q9:  "无" 或 "MCI" → 继续;  "痴呆" → 终止;  "其他" → 继续 + 触发 Q9a + 标记人工审核
Q10: "可以" 或 "1-2天调整" → 继续;  "不确定" 或 "不太可能" → 终止
Q11: "愿意" → 继续;  "不愿意" → 终止
Q14: "同意" → 继续;  "不同意" → 终止
```

### 人工确认清单

自动筛选通过后，以下参与者需额外人工确认：

| 触发条件 | 确认内容 | 确认方式 |
|----------|----------|----------|
| Q7 选"中度" | 音量调大后能否听清语音指导 | Day 1 电话确认 |
| Q8 选"中度" | 字体调大后能否正常使用手机 | Day 1 电话确认 |
| Q9 选"其他" | Q9a 补充的具体疾病是否影响参与 | 电话确认 + 必要时咨询医生 |
| Q10 选"1-2天调整" | 说明灵活补完规则，确认可接受 | 入选电话说明 |
| Q6 选"偶尔帮助" + Q13 选"没有" | 确认是否有其他辅助途径 | 入选电话确认 |

### 通过标准（总结）

符合以下所有条件者入选：
- Q3: 60-80 岁
- Q6: 可以独立或偶尔辅助使用智能手机
- Q7: 非重度听力下降
- Q8: 非重度视力下降
- Q9: 非痴呆/阿尔茨海默病诊断（"其他"需人工审核通过）
- Q10: 可以保证参与（含灵活调整）
- Q11: 愿意接受随访
- Q14: 同意知情同意

### 优先级排序
如报名人数 >50 人，按以下优先级筛选：
1. MoCA 分数 18-25 分 (MCI 范围) — 优先
2. 年龄 65-75 岁 — 次优先
3. 能保证全程参与 (Q10 选"可以") — 优于"1-2天调整"
4. 有亲友协助 (Q13 选"有") — 加分项

### 分层随机化
入选后按以下维度分层：
- 年龄：60-69 岁 / 70-80 岁
- MoCA: 18-21 分 / 22-25 分 (或未知)
- 性别：男 / 女
- 教育：初中及以下 / 高中及以上

每层内随机分配至 A/B 组，确保两组基线均衡。
