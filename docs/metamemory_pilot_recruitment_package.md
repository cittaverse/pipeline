### 筛选逻辑

```
纳入标准:
- 年龄 ≥ 60 岁
- 居住地：杭州 (西湖区/拱墅区优先)
- 能独立使用智能手机 (或有家属协助)
- 愿意分享个人故事
- 有充足时间参与 (2 周 4 次会话)

排除标准:
- 记忆力下降已影响日常生活 (可能 MCI/痴呆，需专业评估)
- 无法使用智能手机
- 不愿意分享故事
- 时间不足
- 严重听力/语言障碍
- 正在服用精神类药物
- 严重抑郁/焦虑症状

分层标准 (用于随机分组):
- 年龄：60-69 / 70-79 / 80+
- 性别：男 / 女
- 参与方式：社区/家中/远程
```
