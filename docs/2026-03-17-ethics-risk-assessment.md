## 四、不良事件报告流程

### 4.1 不良事件定义

- **不良事件 (AE)**: 研究期间发生的任何不利医学事件
- **严重不良事件 (SAE)**: 导致死亡、危及生命、需住院、永久残疾的事件

### 4.2 报告流程

```
发现不良事件
    ↓
24 小时内报告研究协调员
    ↓
研究协调员评估严重程度
    ↓
├── 轻度 AE → 记录在案，继续观察
├── 中度 AE → 报告 PI，暂停干预，评估后决定
└── 重度 AE/SAE → 立即报告 PI + 伦理委员会，终止干预
    ↓
填写《不良事件报告表》
    ↓
伦理委员会审查 (SAE 需 7 天内)
```

### 4.3 不良事件记录表

| 字段 | 说明 |
|------|------|
| 参与者编号 | 去标识化编号 |
| 发生日期 | YYYY-MM-DD |
| 事件描述 | 详细记录事件经过 |
| 严重程度 | 轻度/中度/重度 |
| 与研究相关性 | 无关/可能相关/肯定相关 |
| 处理措施 | 已采取的措施 |
| 结局 | 恢复/未恢复/后遗症/死亡 |
| 报告人 | 姓名 + 日期 |

## 七、应急预案

### 7.1 技术故障应急

| 场景 | 应急措施 | 责任人 |
|------|----------|--------|
| 系统崩溃 | 切换备用服务器，通知参与者稍后重试 | 技术支持 |
| 数据丢失 | 从备份恢复，通知参与者重新完成当日干预 | 技术支持 |
| 网络中断 | 提供离线模式 (本地缓存，恢复后同步) | 技术支持 |

### 7.2 参与者紧急情况

| 场景 | 应急措施 | 责任人 |
|------|----------|--------|
| 严重情绪危机 | 转介心理热线/医院，暂停干预 | 研究协调员 |
| 身体不适 | 建议就医，记录不良事件 | 研究协调员 |
| 失联 (>3 天) | 电话/短信/家属联系，3 次未果视为退出 | 研究协调员 |

### 7.3 研究暂停/终止

| 触发条件 | 决策人 | 流程 |
|----------|--------|------|
| SAE≥2 例且与研究相关 | PI + 伦理委员会 | 暂停 → 审查 → 决定继续/修改/终止 |
| 招募失败 (D-1 <30 人) | PI | 启动备选方案或终止 |
| 依从性过低 (D4 <30%) | PI | 分析原因，修改方案或终止 |
