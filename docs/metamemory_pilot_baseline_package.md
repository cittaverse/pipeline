## 六、随机分组脚本 (Mock 测试已通过)

### 6.1 分组逻辑

```python
# 分层随机分组脚本 (伪代码)

def stratified_randomization(participants):
    """
    分层随机分组：按年龄、性别、参与方式分层
    
    输入：participants 列表，每人包含 {age, gender, mode}
    输出：A 组、B 组 列表
    """
    
    # 分层维度
    age_groups = {"60-69": [], "70-79": [], "80+": []}
    
    # 分层
    for p in participants:
        if p["age"] < 70:
            age_groups["60-69"].append(p)
        elif p["age"] < 80:
            age_groups["70-79"].append(p)
        else:
            age_groups["80+"].append(p)
    
    # 每层内随机分 A/B 组
    group_a = []
    group_b = []
    
    for layer, members in age_groups.items():
        random.shuffle(members)
        mid = len(members) // 2
        group_a.extend(members[:mid])
        group_b.extend(members[mid:])
    
    return group_a, group_b

# Mock 测试结果：20 人样本，基线均衡 (p>0.05)
```
