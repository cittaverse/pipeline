## 三、研究背景与科学依据

### 3.1 研究背景

**疾病负担**:
- 轻度认知障碍 (MCI) 影响约 15-20% 的 65 岁以上老年人
- 中国 60 岁以上人口约 2.97 亿 (2023 年)，MCI 患者估计 4000-5000 万
- MCI 患者每年约 10-15% 进展为痴呆，但目前缺乏有效的干预手段

**现有干预方法局限**:
- 药物治疗：胆碱酯酶抑制剂效果有限，副作用明显
- 认知训练：效果不稳定，依从性低
- 传统叙事疗法：依赖治疗师经验，难以规模化

**CittaVerse 创新点**:
1. **AI 增强叙事疗法**: 自动事件提取 + 元记忆引导，降低治疗师依赖
2. **个性化引导**: 基于叙事质量评分和情绪唤醒度动态调整引导策略
3. **可规模化**: App 形式，居家可用，降低干预成本
4. **循证设计**: 基于 2026 年 LLM 叙事评分研究和元记忆理论

### 3.2 前期研究基础

**已完成工作**:
- ✅ 叙事评分算法 v0.5 开发完成 (Mock 测试 10/10 通过)
- ✅ 情绪唤醒度检测器集成 (300+ 情绪词库)
- ✅ 元记忆引导问题库构建 (50+ 问题，覆盖 6 维度)
- ✅ ASR 集成方案确定 (Azure/iFlytek，待 API Keys)
- ✅ 知情同意书、风险评估、数据隐私方案起草完成

**待完成工作**:
- 🔲 伦理审批 (本次申请)
- 🔲 受试者招募 (目标 80 人)
- 🔲 Pilot RCT 执行 (8 周干预)
- 🔲 数据分析与论文撰写

### 3.3 科学价值

**理论贡献**:
- 验证 AI 辅助叙事疗法对 MCI 老年人的可行性
- 探索元记忆增强引导 vs 标准引导的效果差异
- 建立叙事质量与认知功能改善的关联模型

**应用价值**:
- 为 MCI 非药物干预提供新方案
- 降低叙事疗法实施门槛，提升可及性
- 为后续大规模 RCT 奠定基础

## 四、研究设计

### 4.1 研究类型

- **设计**: 随机、单盲、两臂、Pilot 随机对照试验
- **随机化**: 1:1 分配 (干预组 vs 对照组)
- **盲法**: 评估者盲 (outcome assessor blinded)
- **对照**: 标准叙事引导 (无元记忆增强)

### 4.2 研究流程图

```
筛选期 (D-7 ~ D0)
    ↓
基线评估 (D1): MoCA, GDS-15, QOL-AD, SWLS
    ↓
随机分组 (1:1)
    ├──→ 干预组 (n=40): CittaVerse 元记忆增强引导
    └──→ 对照组 (n=40): CittaVerse 标准引导
    ↓
干预期 (D2-D7): 每日 1 次叙事，每次 15-20 分钟
    ↓
后测评估 (D8): MoCA, GDS-15, QOL-AD, SWLS, 满意度
    ↓
随访 (D21): MoCA (可选)
    ↓
数据分析
```

### 4.3 干预方案

| 维度 | 干预组 | 对照组 |
|------|--------|--------|
| **叙事主题** | 相同 (6 个标准化主题) | 相同 (6 个标准化主题) |
| **叙事频率** | 每日 1 次，连续 7 天 | 每日 1 次，连续 7 天 |
| **叙事时长** | 15-20 分钟/次 | 15-20 分钟/次 |
| **引导方式** | 元记忆增强引导 (50+ 问题库) | 标准引导 (基础提示) |
| **反馈内容** | 叙事质量评分 + 个性化引导 | 仅叙事质量评分 |
| **情绪检测** | 启用 (动态调整引导策略) | 启用 (仅记录，不调整) |

### 4.4 评估工具

| 评估时点 | MoCA | GDS-15 | QOL-AD | SWLS | 叙事质量 | 满意度 |
|----------|------|--------|--------|------|----------|--------|
| 筛选 (D-7~D0) | ✓ (初筛) | - | - | - | - | - |
| 基线 (D1) | ✓ | ✓ | ✓ | ✓ | - | - |
| 后测 (D8) | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| 随访 (D21) | ✓ (可选) | - | - | - | - | - |

**评估工具说明**:
- **MoCA (Montreal Cognitive Assessment)**: 认知功能筛查，总分 30 分，<26 分为 MCI
- **GDS-15 (Geriatric Depression Scale)**: 老年抑郁量表，总分 15 分，≥5 分为抑郁
- **QOL-AD (Quality of Life-Alzheimer's Disease)**: 生活质量量表，总分 52 分
- **SWLS (Satisfaction With Life Scale)**: 生活满意度量表，总分 35 分
- **叙事质量**: 6 维度自动评分 (事件结构/时间连贯性/因果连接/情感深度/感官细节/信息密度)

## 五、研究对象

### 5.1 纳入标准

1. 年龄 60-85 岁
2. MoCA 评分 18-25 分 (MCI 范围)
3. 能够独立或在协助下使用智能手机/平板
4. 能够用中文进行口头交流
5. 视力/听力矫正后能满足研究要求
6. 签署知情同意书

### 5.2 排除标准

1. 诊断为痴呆 (CDR ≥ 1)
2. 严重精神疾病 (如精神分裂症、双相障碍)
3. 严重神经系统疾病 (如脑卒中后遗症、帕金森病)
4. 严重躯体疾病 (如心衰、肾衰、晚期癌症)
5. 正在参加其他干预性研究
6. 对 AI 系统有明显抵触情绪

### 5.3 招募方式

| 渠道 | 方式 | 目标人数 | 责任人 |
|------|------|----------|--------|
| 社区养老服务中心 | 现场宣讲 + 海报 | 40 人 | 研究协调员 |
| 医院记忆门诊 | 医生转介 | 30 人 | 共同研究者 |
| 线上招募 (微信/社区群) | 招募海报 | 10 人 | 研究协调员 |

**招募材料**:
- ✅ 招募海报 (已设计，待最终审阅)
- ✅ 筛选问卷 (问卷星/腾讯问卷，已设计)
- 🔲 招募话术 (待制定)

### 5.4 退出标准

1. 受试者主动要求退出
2. 出现严重不良事件
3. 依从性过低 (<50% 完成率)
4. 研究者判断继续参与不符合受试者最佳利益

**退出后处理**:
- 已完成的数据仍可用于分析 (除非受试者要求删除)
- 提供适当的转介资源 (如需要)
- 记录退出原因

1. **情绪不适**: 提供心理支持资源，允许随时暂停/退出

5. **依从性保障**: 每日提醒，简化操作流程，提供技术支持热线

## 九、不良事件管理

### 9.1 不良事件定义

**不良事件 (AE)**: 研究期间发生的任何不利医学事件，不一定与研究相关

**严重不良事件 (SAE)**:
- 导致死亡
- 危及生命
- 需要住院或延长住院
- 导致持续或显著的残疾/功能丧失
- 先天性异常/出生缺陷
- 其他重要医学事件

### 9.2 报告流程

```
发现 AE/SAE
    ↓
24 小时内报告 PI
    ↓
PI 评估严重性和相关性
    ↓
SAE → 24 小时内报告伦理委员会
AE → 定期汇总报告
    ↓
采取适当措施 (治疗/暂停/退出)
    ↓
记录和追踪
```

### 9.3 应急预案

| 场景 | 应对措施 | 责任人 |
|------|----------|--------|
| 情绪严重不适 | 暂停干预，提供心理支持资源，必要时转介专业机构 | PI/研究协调员 |
| 技术故障 | 切换备用设备，补做评估，记录故障日志 | 技术支持 |
| 数据泄露 | 立即启动应急预案，通知受影响受试者，报告伦理委员会 | PI/DPO |
| 受试者退出 | 尊重决定，记录原因，提供适当转介 | 研究协调员 |
