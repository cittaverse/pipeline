## 筛选逻辑

### 通过标准
符合以下所有条件者入选：
- Q3: 60-80 岁
- Q6: 可以独立/辅助使用智能手机
- Q7: 非重度听力下降
- Q8: 非重度视力下降
- Q9: 无痴呆/阿尔茨海默病诊断
- Q10: 可以保证参与
- Q11: 愿意接受随访
- Q14: 同意知情同意

### 优先级排序
如报名人数>50 人，按以下优先级筛选：
1. MoCA 分数 18-25 分 (MCI 范围) — 优先
2. 年龄 65-75 岁 — 次优先
3. 能保证全程参与 — 必需
4. 有亲友协助 — 加分项

### 分层随机化
入选后按以下维度分层：
- 年龄：60-69 岁 / 70-80 岁
- MoCA: 18-21 分 / 22-25 分 (或未知)
- 性别：男 / 女
- 教育：初中及以下 / 高中及以上

每层内随机分配至 A/B 组，确保两组基线均衡。
